All API requests must be authenticated using your API key, which you can obtain from your dashboard. Include the API key in the Authorization header of your requests.

Example Request
curl -X GET \
  "https://api.defcon.com/v1/users" \
  -H "Authorization: Bearer your_api_key_here" \
  -H "Content-Type: application/json"

Authentication

Secure your API requests with our robust authentication system

API Key Authentication

All requests to the DEFCON API must be authenticated using an API key. This ensures that only authorized applications can access your data.

1

Get Your API Key

Navigate to your dashboard and generate a new API key in the settings section.

2

Include in Requests

Add the API key to the Authorization header of all your HTTP requests.

3

Keep It Secure

Never expose your API key in client-side code or public repositories.

Authentication Example
// Include API key in request headers
const options = {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  }
};

fetch('https://api.defcon.com/v1/users', options)
  .then(response => response.json())
  .then(data => console.log(data));
