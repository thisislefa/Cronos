Introduction to CSS Grid

Learn the fundamentals of CSS Grid Layout

CSS Grid Layout is a powerful tool that allows you to create complex, responsive web layouts with ease. Unlike Flexbox which is one-dimensional, Grid is two-dimensional, meaning it can handle both rows and columns simultaneously.

To get started with CSS Grid, you first need to define a grid container:

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-template-rows: auto;
  gap: 20px;
}

This creates a grid with three equal columns and automatically sized rows, with 20px gaps between grid items.

Grid Container Properties

Master the properties that control the grid layout

The grid container is the parent element that holds all the grid items. Key properties include:

  • grid-template-columns: Defines the columns of the grid
  • grid-template-rows: Defines the rows of the grid
  • gap: Sets the gaps (gutters) between rows and columns
  • justify-items: Aligns grid items along the inline (row) axis
  • align-items: Aligns grid items along the block (column) axis
/* Example of a complex grid */
.container {
  display: grid;
  grid-template-columns: 200px 1fr 300px;
  grid-template-rows: 100px auto 150px;
  gap: 15px;
  justify-items: center;
  align-items: start;
}

Grid Item Properties

Control how individual items behave within the grid

Grid items (direct children of the grid container) can be positioned and sized using specific properties:

  • grid-column: Specifies which column lines the item spans
  • grid-row: Specifies which row lines the item spans
  • grid-area: Shorthand for grid-row-start, grid-column-start, grid-row-end, and grid-column-end
  • justify-self: Aligns the item inside its cell along the inline axis
  • align-self: Aligns the item inside its cell along the block axis
/* Positioning items in the grid */
.header {
  grid-column: 1 / 4;
  grid-row: 1;
}

.sidebar {
  grid-column: 1;
  grid-row: 2;
}

.main-content {
  grid-column: 2 / 4;
  grid-row: 2;
}

Creating a Layout

Build a complete webpage layout using CSS Grid

Now let's create a complete webpage layout with header, sidebar, main content, and footer:

.layout {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
  grid-template-rows: 80px 1fr 60px;
  grid-template-columns: 250px 1fr 1fr;
  min-height: 100vh;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

This creates a clean, semantic layout that's easy to understand and maintain.

Responsive Grid

Make your grid layouts adapt to different screen sizes

CSS Grid works beautifully with media queries to create responsive layouts:

/* Default layout for larger screens */
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 20px;
}

/* Tablet layout */
@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr 1fr;
  }
}

/* Mobile layout */
@media (max-width: 480px) {
  .container {
    grid-template-columns: 1fr;
  }
}

You can also use auto-fit and minmax() for more fluid responsive grids:

.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 20px;
}
